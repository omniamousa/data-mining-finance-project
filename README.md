# Data Mining Project – Finance Domain

## Project Overview

This project focuses on analyzing customer financial behavior using data mining techniques.

##  Objective

To identify risky customers, analyze transaction patterns, and support decision-making.

##  Dataset

* Customer dataset (387 records)
* Transaction dataset (9611 records)

##  Techniques

* Clustering (Customer Segmentation)
* Classification (Risk Prediction)
* Regression
* Anomaly Detection

##  Tools

* Python
* Pandas
* Scikit-learn
